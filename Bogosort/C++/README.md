# Bogosort
Bogosort is an joke algorithm which sorts an unsorted list of items from (smallest-biggest/biggest-smallest).
it does this by randomly swapping values and checking if the array is sorted now

Example:

| input | output |
|  :-:  |  :-:   |
| [1, 3, 2]    | [1, 2, 3]    |
| [1, 0, -1, 2] | [-1, 0, 1, 2] |
