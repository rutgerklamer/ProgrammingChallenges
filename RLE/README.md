# RLE (Run-length encoding)
RLE is a technique in which we compress multiple of the same characters in a row

Example:

| normal input/output | encoded input/output |
|  :-:  |  :-:  | 
| AABB | 2A2B |
| ABBCCCDDDDEEEEE | 1A2B3C4D5E |
| WWAARR | 2W2A2R|
