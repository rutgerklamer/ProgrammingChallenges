# Comma quibbling
Comma quibbling seperates a given input of strings by a ',' or a 'and'

Example:

|input| output |
|:-:|  :-:   |
|[ "ABC", "DEF" ] |   { ABC, DEF }  |
|[ "ABC", "DEF", "G" ] |   { ABC, DEF and G }  |
