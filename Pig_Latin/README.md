# Pig Latin
Pig latin takes the first two characters of a word and pastes those at the end of the word and add ay behind it

Example:

|  input  |  output   |
|   :-:   |    :-:    | 
| trash   | ashtray   |
| preview | eviewpray |
| banana  | ananabay  |
