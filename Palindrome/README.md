# Palindrome
This program checks if a word is a palindrome or not.
A palindrome is a word that is the same if it's reversed

Example:

| input | output |
|  :-:  |   :-:  |
| racecar| *true* |
| tacocat| *true* |
| pinkfloyd | *false* |
