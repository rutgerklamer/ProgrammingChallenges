# Bubblesort
Bubblesort is an algorithm which sorts an unsorted list of items from (smallest-biggest/biggest-smallest)

Example:

| input | output |
|  :-:  |  :-:   |
| [1, 3, 2]    | [1, 2, 3]    |
| [1, 0, -1, 2] | [-1, 0, 1, 2] |
