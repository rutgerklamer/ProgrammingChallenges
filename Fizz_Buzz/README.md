# Fizz Buzz
This program prints the numbers from 1-100, But for every multiple of 3 we print Fizz, 
For every multiple of 5 we print Buzz and for both a multiple of 3 and 5 we print FizzBuzz.

Example:

| input | output |
|  :-:  |  :-:   |
|  1    |   1    |
|  3    |   Fizz |
|  5    |   Buzz |
|  15   |FizzBuzz|
