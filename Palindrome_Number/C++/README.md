# Palindrome number
A palindrome number is a number that has the same value if it's reversed

Example:

|  input  |  output   |
|   :-:   |    :-:    | 
| 111   | *true*   |
| 15651 | *true* |
| 568  | *false*  |

Widipedia for [Further info](https://en.wikipedia.org/wiki/Palindromic_number)
